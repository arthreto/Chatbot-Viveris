
### 1. Prérequis

- PHP 7.4 ou supérieur
- Extension **PDO SQLite** activée
- Serveur web (Apache/Nginx)
- Compte Google Cloud Platform (pour OAuth si vous souhaitez)
- Clé API OpenAI (Groq)

### 2. Configuration

Modifiez le fichier `config/config.php` si besoins (recommendé)
